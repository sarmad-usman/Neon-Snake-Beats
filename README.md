## 🖥 Run Locally

### 📌 Prerequisites

* Node.js (v18+ recommended)
* npm or yarn

---

### ⚙️ Setup Instructions

```bash id="9k2m1x"
git clone https://github.com/your-username/neon-snake-beats.git
cd neon-snake-beats
npm install
```

---

### 🔐 Environment Setup

Create a `.env.local` file:

```env id="env123"
GEMINI_API_KEY=your_gemini_api_key_here
```

---

### ▶️ Start Development Server

```bash id="dev456"
npm run dev
```

Then open:
http://localhost:3000
